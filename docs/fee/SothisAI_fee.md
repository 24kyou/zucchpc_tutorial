# 曙光平台计费规则
## 计费模式
曙光平台采取预计费模式。每位用户预先拥有10000（1W）小时数，在用户申请容器/作业时，根据用户申请容器/作业所需小时数（<span style="background:red">用户申请的CPU和GPU的**核数**×GPU/CPU每一核的**小时费用**×用户申请的容器/作业的**时间**</span>）对用户剩余小时数进行预先扣除，如果用户剩余小时数小于用户申请的小时数，则容器/作业创建失败。
## 计费价格

| 名称    | 价格（费用/核*时） |
| :-------------- | :-----------:|
| CPU      | 1       |
| GPU：T4   | 1        |
