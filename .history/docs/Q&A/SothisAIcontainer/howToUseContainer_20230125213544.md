# 曙光AI平台容器实例和notebook服务区别及使用方式。 
## notebook服务和容器实例服务的区别。  

相同之处：曙光AI平台上的容器和notebook本质上是一样，底层都是用docker实现。

不同之处：
1. notebook和容器实例的区别在于nootebook所提供的资源是先配置好的，而容器实例的资源可以自由调整。
2. notebook服务界面提供的**操作**选项并不全面，更多操作需要到**容器服务->人物列表**
